Task 006.01 – Build/Run Docs & Shader Compile Helper

Document how to build/run and compile shaders deterministically.

Docs content

  • Prereqs: Go, Vulkan runtime, glslc, GLFW/X11 dev packages.
  • Build: go build ./...
  • Run: go run . (mention VK_VALIDATION=0 to disable validation).
  • Controls: ESC exits, SPACE pauses rotation.

Shader compile helper

glslc shaders/cube.vert -o shaders/vert.spv
glslc shaders/cube.frag -o shaders/frag.spv
glslc shaders/overlay.vert -o shaders/overlay_vert.spv
glslc shaders/overlay.frag -o shaders/overlay_frag.spv

Provide a script/Makefile target to run these commands and fail on error.
