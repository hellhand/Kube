Task 001 – Vulkan Go Setup & Prerequisites

Install tools, verify drivers, and prepare your system for Vulkan + Go.

What to do
  • Install Go 1.24+ (go version).
  • Install Vulkan loader/driver + dev headers (via SDK or distro packages).
  • Install X11/GLFW dev packages so GLFW can build (e.g., libx11-dev, libglfw3-dev on Debian/Ubuntu).
  • Install glslc (from the Vulkan SDK) to compile shaders.

Verify Vulkan runtime

vulkaninfo | head
glslc --version
go version

If vulkaninfo fails, install or fix the GPU driver/ICD first.

Validation layers

  • If the SDK is installed, validation layers are usually in place. If missing, you can run with VK_VALIDATION=0 to skip them.
  • Environment hints (if needed): VK_LAYER_PATH to point to layer JSON, VK_ICD_FILENAMES for ICD JSON.

Why this matters

All later steps assume Vulkan initializes cleanly and shaders can be compiled. Fixing drivers early saves time.

Detailed checklist

Run each of these once and paste the output into your notes so teammates can compare results.

# GPU + ICD detection
vulkaninfo | head -n 25

# Shader compiler
glslc --version

# Go + cgo toolchain
go env GOPATH GOROOT CGO_ENABLED CC CXX

# Distro packages (Ubuntu/Debian example)
sudo apt-get install libvulkan-dev vulkan-utils mesa-vulkan-drivers \
  libx11-dev libxcursor-dev libxrandr-dev libxinerama-dev libxi-dev \
  libglfw3-dev libxxf86vm-dev

If any command fails, fix it before advancing. Vulkan issues cascade and are much harder to diagnose once swapchains and pipelines enter the picture.
