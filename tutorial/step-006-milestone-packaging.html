Task 006 – Packaging, Documentation, and QA

Provide build/run docs, shader compile steps, and basic smoke checks.

Milestone goal

Have reproducible build/run instructions, shader compilation commands, and minimal validation/smoke coverage.

Quick commands

go build ./...
glslc shaders/cube.vert -o shaders/vert.spv
glslc shaders/cube.frag -o shaders/frag.spv
glslc shaders/overlay.vert -o shaders/overlay_vert.spv
glslc shaders/overlay.frag -o shaders/overlay_frag.spv
VK_VALIDATION=0 go run .   # disable validation if needed

Include these in docs or scripts/Make targets.

What to produce

  • README/docs with requirements, build/run steps, and env toggles.
  • Script or Make target to compile GLSL to SPIR-V.
  • Smoke-check flow: start app, render cube/overlay, exit cleanly (ESC).

Verification script example

#!/usr/bin/env bash
set -euo pipefail
go build ./...
make shaders     # or invoke glslc lines manually
VK_VALIDATION=0 go run . --smoke
echo "Smoke OK"

Drop this into scripts/smoke.sh and reference it in the docs so contributors have a single, reliable entry point to test changes.
